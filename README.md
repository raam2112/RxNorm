# 💊 Drug Search & Tracker API with RxNorm Integration

A secure Laravel 12 application that allows users to register/login, search drugs using the National Library of Medicine's **RxNorm API**, and maintain a private list of tracked medications.

---

## 🚀 Features

- 🔐 **User Authentication** (Laravel Breeze / Sanctum)
- 🌐 **RxNorm API Integration**
- 🧪 **Validation & Unit Testing** (90%+ coverage)
- 📥 **Add/Delete Tracked Drugs**
- 👤 **Private Medication List**
- 🌍 **Rate Limiting & Caching**
- ✅ **RESTful API + Blade UI Views**

---

## 📦 Tech Stack

- Laravel 12
- Laravel Breeze (Frontend: Blade)
- RxNorm API
- Guzzle HTTP Client
- Sanctum (for API tokens)
- PHPUnit (for testing)
